### 🎮 Features

- 📚 5 topics with 4 questions each
- 💰 Questions vary in difficulty and prize value
- 🛡️ Safety net system — start with $700, doubles with every correct $5000 answer
- ⏱️ 20 seconds to answer each question
- ❌ One wrong answer ends the game
- 🏆 Answer everything correctly to keep the full prize pool

<br>

### 🚀 How to Run

1. Make sure **Java** is installed on your computer
2. Download or clone this repository
3. Open the project in **Eclipse IDE**
4. Run the main file
